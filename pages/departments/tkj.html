Computer & Network Engineering (TKJ)

Build and secure the digital infrastructure of tomorrow

Program Description

The Computer & Network Engineering program (TKJ) at SMK Telkom Malang provides comprehensive training in computer networking, system administration, and cybersecurity. Students learn to design, implement, and maintain network infrastructure while developing skills in network security and troubleshooting.

What You'll Learn

  • Network Infrastructure Design
  • System Administration
  • Network Security
  • Cloud Computing

Career Opportunities

  • System Administrator
  • Security Specialist
  • Network Engineer
