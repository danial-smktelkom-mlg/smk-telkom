Software Engineering (RPL)

Shape the future through innovative software development

Program Description

The Software Engineering program (RPL) at SMK Telkom Malang is designed to provide students with comprehensive knowledge and skills in software development. Our program covers various aspects including system analysis, software design, programming, and testing. Students will learn to use various programming languages and development tools to create innovative applications.

What You'll Learn

  • Web Development (Frontend & Backend)
  • Mobile App Development
  • Database Management Systems
  • Software Development Life Cycle

Career Opportunities

  • Software Developer
  • Web Developer
  • Mobile App Developer
  • System Analyst
